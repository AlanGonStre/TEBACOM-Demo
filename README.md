# TEBACOM – Tablero Académico (Demo Académica)
## Descripción
Proyecto demo (académico) basado en un módulo de tablero académico con gráficas (Google Charts) para:
- Distribución por género
- Registro de lenguas indígenas
- Avance de captura de calificaciones por parcial y grado
> Nota: Este repositorio es una versión **sanitizada**. No incluye credenciales, conexiones ni datos institucionales.
## Tecnologías
- ASP.NET WebForms (.NET Framework)
- C#
- Entity Framework
- SQL Server (LocalDB o instancia local) *(opcional si se usa modo demo)*
- Google Charts
## Requisitos
- Visual Studio 2022
- .NET Framework (4.7.2)
-## Configuración
### Modo Demo (sin base de datos)
1. Abrir la solución en Visual Studio.
2. Ejecutar el proyecto (F5).
3. El sistema mostrará datos simulados para las gráficas.
